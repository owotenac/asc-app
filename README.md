# Welcome to the ASC APP

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```


3. Internal preview build
   ```bash
   npx expo-doctor
   eas build --platform android --profile preview
   ```
